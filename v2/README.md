# Chatbot v2 – Rozszerzona rozmowa

Wersja v2 rozwija funkcjonalność podstawowej wersji chatbota.  
Bot prowadzi krótką rozmowę, zadając użytkownikowi kolejne pytania:

1. O imię  
2. O samopoczucie (dobrze/kiepsko/źle)  
3. O zainteresowanie uniwersum Gwiezdnych Wojen  

W zależności od odpowiedzi bot:
- reaguje emocjonalnie,
- dopytuje o ulubioną postać (jeśli użytkownik lubi Star Wars),
- wyświetla neutralną odpowiedź, jeśli użytkownik nie interesuje się tematem.

Wersja v2 nadal nie posiada:
- pętli rozmowy,
- słów kluczowych,
- kontekstu.

Jest to rozszerzona, ale nadal liniowa forma dialogu.
