# Chatbot v1 – Podstawowa wersja

To jest pierwsza i najprostsza wersja chatbota.  
Bot zadaje użytkownikowi dwa pytania:

1. O imię
2. O potwierdzenie, czy imię jest prawdziwe

Na podstawie odpowiedzi bot wyświetla jedną z trzech reakcji:
- pozytywną,
- negatywną,
- informującą o niepoprawnej odpowiedzi.

Wersja v1 nie zawiera:
- pętli rozmowy,
- słów kluczowych,
- rozbudowanej logiki,
- kontekstu rozmowy.

Jest to punkt wyjścia do dalszego rozwoju chatbota.
