# Chatbot v4 – Zaawansowany chatbot z kontekstem

Wersja v4 jest najbardziej rozbudowaną wersją chatbota w tym repozytorium.  
Bot rozpoznaje różne kategorie wypowiedzi i zapamiętuje kontekst rozmowy, dzięki czemu może udzielać odpowiedzi zależnych od wcześniejszych wiadomości.

Rozpoznawane kategorie:
- powitania,
- pytania o samopoczucie,
- emocje pozytywne,
- emocje negatywne,
- tematy związane z Gwiezdnymi Wojnami,
- pożegnania.

Nowości w tej wersji:
- zmienna `current_topic`, która pozwala kontynuować temat,
- odpowiedzi zależne od kontekstu,
- bardziej naturalne reakcje emocjonalne,
- subtelna odpowiedź domyślna, gdy bot nie zna tematu.

Wersja v4 stanowi najbardziej kompletną i elastyczną formę chatbota w tym projekcie.
