# Chatbot v3 – Chatbot oparty na słowach kluczowych

Wersja v3 wprowadza znaczącą zmianę w sposobie działania chatbota.  
Bot działa w pętli i reaguje na słowa kluczowe wpisane przez użytkownika.

Rozpoznawane kategorie:
- powitania,
- pytania o samopoczucie,
- odniesienia do Gwiezdnych Wojen.

Jeśli bot nie zna odpowiedzi, wyświetla komunikat informujący, że jeszcze się uczy.

Nowości w tej wersji:
- pętla rozmowy,
- obsługa wielu różnych wypowiedzi,
- prosty system dopasowywania słów kluczowych.

Wersja v3 nie posiada jeszcze pamięci kontekstu.
